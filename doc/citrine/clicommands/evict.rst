evict
--------

.. code-block:: text

  evict [--fsid <fsid>] [-f|--force] <path>|fid:<fid-dec>]|fxid:<fid-hex> [<path>|fid:<fid-dec>]|fxid:<fid-hex>] ...
    Evicts all disk replicas of the given files separated by space
    This command requires write and p acl flag permission
